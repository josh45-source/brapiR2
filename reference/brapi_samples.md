# List Samples

List Samples

## Usage

``` r
brapi_samples(con, ...)
```

## Arguments

- con:

  A
  [`brapi_connection()`](https://josh45-source.github.io/brapiR2/reference/brapi_connection.md)
  object.

- ...:

  Additional query parameters (e.g. `studyDbId`, `germplasmDbId`,
  `observationUnitDbId`).

## Value

A tibble with one row per sample.
