# List Seed Lots

List Seed Lots

## Usage

``` r
brapi_seed_lots(con, ...)
```

## Arguments

- con:

  A
  [`brapi_connection()`](https://josh45-source.github.io/brapiR2/reference/brapi_connection.md)
  object.

- ...:

  Additional query parameters.

## Value

A tibble with one row per seed lot.
