# List Seasons

List Seasons

## Usage

``` r
brapi_seasons(con, ...)
```

## Arguments

- con:

  A
  [`brapi_connection()`](https://josh45-source.github.io/brapiR2/reference/brapi_connection.md)
  object.

- ...:

  Additional query parameters (e.g. `year`).

## Value

A tibble with one row per season.
