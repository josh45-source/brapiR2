# List Germplasm

List Germplasm

## Usage

``` r
brapi_germplasm(con, ...)
```

## Arguments

- con:

  A
  [`brapi_connection()`](https://josh45-source.github.io/brapiR2/reference/brapi_connection.md)
  object.

- ...:

  Additional query parameters (e.g. `commonCropName`, `germplasmName`,
  `studyDbId`).

## Value

A tibble with one row per germplasm accession.
