# List Call Sets (Samples with Genotype Data)

List Call Sets (Samples with Genotype Data)

## Usage

``` r
brapi_call_sets(con, ...)
```

## Arguments

- con:

  A
  [`brapi_connection()`](https://josh45-source.github.io/brapiR2/reference/brapi_connection.md)
  object.

- ...:

  Additional query parameters.

## Value

A tibble with one row per call set.
