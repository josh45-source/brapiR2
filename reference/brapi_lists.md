# List Generic Lists

List Generic Lists

## Usage

``` r
brapi_lists(con, ...)
```

## Arguments

- con:

  A
  [`brapi_connection()`](https://josh45-source.github.io/brapiR2/reference/brapi_connection.md)
  object.

- ...:

  Additional query parameters (e.g. `listType`).

## Value

A tibble with one row per list.
