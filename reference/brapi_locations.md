# List Locations

List Locations

## Usage

``` r
brapi_locations(con, ...)
```

## Arguments

- con:

  A
  [`brapi_connection()`](https://josh45-source.github.io/brapiR2/reference/brapi_connection.md)
  object.

- ...:

  Additional query parameters (e.g. `locationType`).

## Value

A tibble with one row per location.
